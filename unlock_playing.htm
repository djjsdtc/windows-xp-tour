播放音乐、视频和 CD

是否喜欢听音乐或者看电视？Windows XP 具有 Windows Media Player，所以很容易完成以下功能：

  • 收听 Internet 广播
  • 从 Web 上下载音乐或电影。
  • 收看流畅的视频广播。
  • 播放自己的 CD。

使用 Windows Media Player 可以很容易地创建自己的可以自定义自动唱片点唱机。只需：

  1. 将个人 CD 复制到计算机或者从 Internet 下载音乐。
  2. 按想要的顺序排列选择项。
  3. 存储播放清单。

计算机可以在您喜欢的任何时候播放选择项 - 按照选择的顺序一个接着一个播放。甚至可制作自己的 CD！
